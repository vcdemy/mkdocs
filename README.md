# MkDocs使用簡介

MkDocs是靜態網站產生工具，他可以將我們寫好的markdown文件迅速地轉換成一個網站。

## 安裝

```bash
$ pip3 install mkdocs
```

## 基本使用說明

### 建立新的專案

```bash
# 在當下的目錄建立新專案
mkdocs new .
```

```bash
# 建立一個新專案，並將專案的資料夾命名為projectname
mkdocs new projectname
```

### 啟動伺服器來檢視網站

```bash
mkdocs serve
```

### 建立說明網站

```bash
mkdocs build
```

### 佈署到github pages

```bash
mkdocs gh-deploy
```
